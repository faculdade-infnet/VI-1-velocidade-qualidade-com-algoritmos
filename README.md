## Commmit Patterns
| Symbol  | Type       | Use              |
| ------- | ---------- | --------------------------------------- |
| 🔧      | `build`    | csproj, NuGet, build                    |
| 👷      | `ci`       | CI/CD Pipeline   |
| 🧹      | `chore`    | Routine maintence task                  |
| 📚      | `docs`     | Documentation    |
| ✨      | `feat`     | Add a new functionality                 |
| 🐛      | `fix`      | Fix a bug        |
| 🚚      | `move`     | Move/rename files |
| ⚡      | `perf`     | Performance      |
| ♻️      | `refactor` | Improves code without changing behavior |
| 🔥      | `remove`   | Remove code      |
| 🎨      | `style`    | Formatting/code without change logic    |
| 🧪      | `test`     | Tests            |
| 🚧      | `wip`      | Work in progress |
